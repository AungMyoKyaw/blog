တာ့တာ့ medium [ ghost static ]

2021 နှစ်ချုပ် ရေးမလို့ ဝင်လိုက်တာ ဝင် လို့ မရဖြစ်နေတာ ရယ်။​ကိုယ် ပိုင် control အကုန် သုံးချင်တာ ရယ်။ static site ပြန်ပြောင်းမလို့ စိတ် ကူး ရလာတယ်။

medium မှာ ရေး ဖြစ်တာ က editor ကို ကြိုက်လို့ ။ ‌‌note ထဲ ဖြစ်ဖြစ် ရေးထားပြီးတော့ paste ချပြီးတော့ လည်း မရေးချင်လို့။‌‌အစ ပထမ တုန်းက တော့ static site generator သုံးတယ်။​ hugo သုံးဖြစ်တယ်။‌‌WYSIWYG နဲ့ သုံးတာ ကြိုက်တယ်။‌‌medium ကို ရောက်လာတယ်။
နောက်ပိုင်း custom domain ပေးသုံးတယ် အဆင်ပြေလာတယ်။​

2021 နှစ်ချုပ် ရေးမလို့ ဝင်လိုက်တာ ဝင် လို့ မရဖြစ်နေတာ ရယ်။​ကိုယ် ပိုင် control အကုန် သုံးချင်တာ ရယ်။ static site ပြန်ပြောင်းမလို့ စိတ် ကူး ရလာတယ်။ ဒါပေမယ့် note ထဲရေးပြီးတော့ local မှာ build ရတာ တင် ရတာ စိတ် ညစ်လို့ နောက် တမျိုး သုံးမလို့ စဉ်းစားတော့ ghost သုံးချင်တယ်။

ပြဿနာက static က တော့ အလကား တင်ဖို့ နေရာ ပေါတာ ။ ကိုယ် က ခဏ​လေးပဲ​ရေးတာ ဆိုတော့ server ဖိုး လည်း ကုန်တာ မတန်လို့ ။ တဝက် စီ ပဲ​ သုံးတော့မယ်။

အခု က ghost ကို local မှာထားပြီးတော့ စာ ရေး ဘာညာ လုပ် မယ်။

ပြီးရင် local က ဟာ ကို static အနေ နဲ့ အကုန် crawl ပြီးတော့ တင် မယ်။

အဲ့တာ က အဆင်ပြေတယ်။ ကိုယ့် လက် ထဲမှာ အကုန် control လည်း ရတယ်။ editor ကောင်းကောင်းလည်း သုံး လို့ ရတယ်။

gssg က index file ကို crawl မလုပ်တဲ့ error ရှိတယ် အဲ့တာ က pull request တယောက် ပို့ထားတယ်။ အဲ့ ဟာကို ပဲ install လုပ်သုံး ရင် အဆင်ပြေတယ်။

နောက်တခါတော့ medium ကို ပြန်ပြီးတော့ မသွားတော့ပါဘူး။ ကိုယ့် နေရာမှာပဲ ကိုယ် ထားတော့မယ်။

အခု တော့ site ကို github page ကနေ cloudflare pages ကို ရွှေ့ လိုက်တယ်။

site repo ကတော့ အခု နေရာမှာ ကြည့်လို့ ရတယ်။
