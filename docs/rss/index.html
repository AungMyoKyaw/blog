အစ တုန်းက တော့ magic keyboard သုံးနေတာ နောက်တော့ ကိုယ် ရိုက်တဲ့ ဒဏ် မခံ နိုင်လို့ enter key က နေ စ လစ်သွားတာ ။ အစ ကတော့ karabiner နဲ့ ပြန် map ပြီးတော့ ctrl + m ဆိုရင် enter ဆိုပြီးတော့ သုံးသေးတယ် ။ နောက်တော့ မှတ်တော့ မမှတ်မိတော့ဘူး ဘာကြောင့်လဲ မသိဘူး ရယ် လုံး ဝ လစ် သွား လို့ တိုင်တွေ ပတ်တော့တာ ပဲ အလုပ်လုပ် ရတာ အဆင်ပြေမနေဘူး တွေ ဘာ တွေ ဖြစ်ကုန်ရော ။ နောက် ကီးဘုတ် တခု ဝယ်ဆိုလည်း လွယ် လွယ် ကူကူ ဝယ်လို့ ရတဲ့ နေရာ မဟုတ်တော့ ရုံးက ရှိတဲ့ AUKEY လေး တခု ပို့ပေးတယ် ။ အလုပ်ကတော့ လုပ်လို့ ရတာပေါ့ ။​ ကီးဘုတ် ဆိုတာ စာ ထွက် ရင်ပြီး ရောလို့ ဆိုဘေမယ့် အရသာ ခြင်းမတူဘူး လေနော့ ။ အဲ့တော့ အဆင်တော့ပြေပါ ရဲ့ လက် မှာ ဟာ တာတာ ကြီး ပေါ့ ။ သူ့ဟာ က ဆေးကြီး option < = > command ကို ပြန် ပြီးတော့ swap ရတယ် ။ သူ ဟာ ကတော့ အဆင်ကို မပြေဘူး ။ နောက်တော့ Logitech လေး တခု ပို နေတာ နဲ့ ပြန်ပြီးတော့ အဲ့တာလေး သုံးဖြစ်တယ် အဲ့မှာတော့ အတော်လေး ကို အဆင်ပြေသွားပါပြီ။

နောက်တော့ တနေ ကုန် သုံး ဖြစ်တဲ့ ကောင် အဖို့တော့ ဒါလေးတော့ကောင်းကောင်းလေး တော့ဖြစ်သင့်တာ ပေါ့ ဘာ ညာ psycho ဖောက် လာ ရော ။ အရင်တုန်းကလည်း keychron k2 review တခု ဖတ်ထားတော့ အဲ့ နာမည် ပဲ ကြားဘူးတာ ဆိုတော့ အဲ့ အထဲ က ပဲ လုပ်မယ်လို့ စိတ်ကူးထားတာ ။

ပုံမှန် က ထူထူ ကြီးတွေ ဆိုတော့ palm rest က လို တယ် ။ အိုက်မှာ k1 v4 က ပါးပါးလေးဖြစ်တာ ဆိုတော့ သဘော ကျသွားတယ်။ palm rest မလို တဲ့ keyboard ပေါ့ လေ။

ကျန်တဲ့ ဟာ က တော့ ကောင်းလားမကောင်းလားဆိုတာကို သူများတွေ hype တာ လိုက်ကြည့်ပြီးတော့ ဒီမှာ လည်း ဝယ် လို့ ရတယ် ဆိုတော့ ဆော်လိုက်တာ ပဲ ။

switch ကတော့ ခက်ခက် ခဲခဲ မဟုတ်ဘူး clicky sound က အရမ်းများနေရင်လည်း အိမ် က ထရိုက်မှာ ဆိုးလို့ အလယ် အလတ် ကိုပဲ သုံးဖြစ်တယ်။ အမှန် ကတော့ type သုံးခု လုံး ယှဉ် ရိုက်ပြီးတော့ ငါတော့ ဘာ နဲ့ ကိုက်လဲ ဆိုပြီးတော့ ဆုံးဖြတ်တာကတော့ အကောင်းဆုံးပေါ့ ။ အခုတော့ ကြား ကျတဲ့ brown switch ပဲ သုံးလိုက်တယ်။

ကိုယ် ပဲ ခံ စား မှုက အားနည်းနေလာတော့ မသိဘူး သူ့ ဟာ က အလယ် ခေါင်မှာ ထစ်ထစ် ထစ်ထစ် ဖြစ်တယ် ဆိုတာကို ဘယ်လို မှမခံ စား နိုင်ဘူး ။​

စစချင်းကတော့ အရမ်းသုံးချင်နေတဲ့ သူဆိုတော့ သုံး ရက် လောက်ကို မနည်း အောင့်ပြီးတော့ စောင့် ရတယ် သောကြာ နေ့ က မှာ တာ တနင်္လာ နေ့တော့ ရောက်လာတယ်။

အဲ့တော့စစ ချင်တော့ အရမ်းကြီးတော့ နေသားမကျတော့ ကိုယ့်အတွက် ပိုသွားတာ က click click click မြည်တာရယ် မီးလင်းတာ ရယ် ပဲ ရှိတယ်။​

နောက် နှစ် ရက် လောက် လည်း နေရော ကိုယ့် ကောင်က အစွမ်းတွေ ထ ပြ လာတယ် ။ လက် တွေ ကိုက်လာ တာ ။ လခွမ်းသူ တို့ က အားရှိလို့ ရိုက်နိုင် တာ က ကိစ္စ မရှိဘူး ။ ဒီ က ကောင် က ပုံမှန်ထက် အား က စိုက် နေရတော့ စောက် ရမ်း ကို တိုင်တွေ ပတ် နေရော ။​

vim ကိုပဲ တစိုက် မက်မက် သုံးဖြစ်တာ ၁၁ လ လောက် တော့ ရှိသွားပြီ။ config ချတယ် ဆိုတာလည်း ဟို ကကူးလိုက် ဒီ က ကူးလိုက် plugin ပေါင်း သောင်း ခြောက် ထောင်လောက်ကို install လုပ် ရင်းနဲ့ vim က sublime လိုလို ဘာလိုလို ဖြစ်လာအောင် လုပ် လိုက်တာ ပါပဲ။ theme chanage ပြီးတော့ syntax highlight ဖွင့်ထားလိုက်တော့ အမြင်အားဖြင့် ကြည့်လှသွားတာပါပဲ။

လွန်ခဲ့ တဲ့ လတုန်းက static site လေးပြန်လုပ်မယ် ဆိုတဲ့ စိတ်ကူးပေါ်လာတယ်။ အဲ့တော့ ဗမာလို့ ဘယ်မှာ ရိုက်ဖြစ်မလဲ ဆိုတော့ ပြဿနာ ပေါ်လာတယ်။ ကဲ အဲ့တော့ emacs ပဲ ဆိုပြီးတော့ emacs ပြောင်းသုံးမယ် ကြိုးစားတယ်။ အခု လောလော ဆယ် ကိုယ် အသားကျ နေတဲ့ vim + tmux ကို စွန့်လိုက်မယ် လို့ စိတ်ကူးလိုက်တယ်။ အဲ့တော့ emacs ပြောင်းမယ် ကြိုးစားတော့ သူ့ ရဲ့ စိတ်ပျက် စရာ ကောင်းတဲ့ key binding တွေနဲ့ အရင် တွေ့တော့တာပါပဲ။

vim စသုံးကာစ က jkhl ကို ခက်လှတယ် မှတ် တယ် သူ့ ရဲ့ ctrl key ကြီးနဲ့ တွဲ ကိုင် ရတာကြီးတွေက အတော်လေး စိတ်ကုန်စရာ ကောင်းတယ်။ နောက် တမျိုး ဒုက္ခ ရောက်တာက vim မှာ ဆို normal mode ရယ် insert mode ရယ် visual mode ရယ် ရှင်းရှင်း လင်းလင်း လေး ရှိတဲ့ ကိစ္စကို အခု emacs မှာ ကျတော့ language syntax တွေ အလိုက်ခွဲထားတယ် ။ နောက် ပြီးတော့ normal mode မရှိတော့ vim မှာ လို normal mode ပြောင်းပြီးတော့ ဟိုပြေးဒီ ပြေးလုပ်လို့မရတာလည်း စိတ်ပျက်စရာပါပဲ။ နောက်တော့ မထူးတော့ဘူး ဒီ အတိုင်းဆို မဖြစ်ဘူး ဆိုပြီးတော့ evil mode ပဲ သုံးလိုက်တယ်။ plugin ပေါင်း သောင်းခြောက်ထောင် ထည့်လိုက် config တွေ လိုက် ကူးချလိုက်တော့ သုံးရတာ အဆင်ပြေသွားတယ်။

spacemacs ဆိုတာ ရှိတာ သိတယ်။ ဒါပေမယ့် အစ ပထမ တုန်းက emacs key binding တွေ နဲ့ပဲ ရအောင် သုံးမယ် လို့ စိတ်တင်းထားတော့ လှည့်ကို မကြည့်ဖြစ်ဘူး။ နောက်တော့ evil mode တောင် သုံးရပြီ ဆိုတော့ မထူးတော့ပါဘူး ဆိုပြီးတော့ spacemacs ကိုပဲ သုံးဖြစ်တယ်။

လူ့စိတ်က မျောက်စိတ် ဆိုတော့ သိပ်မှန်တယ်။ အစတုန်းက static site လေး လုပ်မှ ပဲ blog post တွေ အပတ် တိုင်း တက်ပြီးတော့ လုံးဝ productive ဖြစ်မယ် ဆိုပြီးတော့ emacs ဖက် ရောက်သွားတာ ။ နောက်တော့လည်း မထူးပါဘူး ကွာ ဆိုတဲ့ စိတ်ဝင်လာလိုက်တော့ static site ကိစ္စလည်း တခါတည်း စမုံ တုံး သွားတော့တာပါပဲ။
